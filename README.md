```
OperationExcel是使用 POI 操作Excel文件的案例.
POI官方介绍：
Apache POI api的主要用途是用于文本提取应用程序，如web spider、索引生成器和内容管理系统。
那么，为什么要使用POIFS、HSSF或XSSF呢?
如果您有一个用OLE 2复合文档格式编写的文档(可能是用MFC编写的)，需要用Java读取，那么您应该使用POIFS。
或者，如果需要与运行在Windows平台上的软件进行互操作，您可以使用POIFS来编写OLE 2复合文档格式。
当我们说POIFS是迄今为止最完整、最正确的文件格式实现时，我们并不是在吹牛!
如果需要使用Java (XLS)读取或写入Excel文件，可以使用HSSF。
如果需要使用Java (XLSX)读取或写入OOXML Excel文件，那么应该使用XSSF。
组合的SS接口允许您轻松地使用Java读写各种Excel文件(XLS和XLSX)。
此外，还有一个专门的SXSSF实现，它允许以内存优化的方式编写非常大的Excel (XLSX)文件。
POI官方地址：https://poi.apache.org/
```
